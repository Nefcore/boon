<h1 align="center">boon</h1>

`boon` can find usernames across over 45 social media platforms with concurrency.

## Installation

```
$ git clone https://github.com/Nefcore/boon.git
$ cd boon
$ python3 setup.py install
$ boon nefcore
```

## Usage

Syntax: `boon <username>`
